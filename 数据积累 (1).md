###### 解决Ubuntu/debian添加国内源时报gpg-key错问题

```bash
在添加一个其他的源后，刷新源
sudo apt update
...
GPG error: https://mirrors.tuna.tsinghua.edu.cn/ubuntu bionic-proposed InRelease: The following signatures couldn't be verified because the public key is not available: NO_PUBKEY 3B4FE6ACC0B21F32
...

解决方法：将NO_PUBKEY 后的密钥添加至系统源中
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 3B4FE6ACC0B21F32

sudo  apt update
```


###### sed用法：


###### grep命令

```bash
-c 行数
-i 不区分大小写
-n 显示行号
-v 取反
-r 遍历所有子目录
-A 后面跟数字，过滤出符合要求的行以及下面n行
-B 同上，过滤出符合要求的行以及上面n行
-C 同上，同事过滤出符合要求的行以及上下各n行
grep ‘[0-9]’ filename 查看文件中包含数字0-9的行
grep -v ‘[0-9]’ filename 查看文件中不包含数字的行
grep -v ‘^#’ filename 查看文件中不带#号的行
grep ‘[^0-9]’ filename 查看非0-9的字符，即包含0-9的字符不被高亮显示
grep ‘[0-9]’ filename 筛选非数字开头的行
grep ‘r.o’ filename 筛选包含“r与o”的行 “.”为任意字符
grep -n ‘o*o’ filename 前面表示0-n次出现，后面表示前面匹配的字符应该与后面的一致。及多次出现该字符都会被匹配进来。
grep '.’ passwd filename 匹配任意符号任意字符
grep ‘o{2}’ filename 表示筛选带有两个o字符的行
grep -n ‘o{0,3}’ filename 表示筛选0-3个带有o字符的行
egrep命令：
egrep -q  取反
egrep -n  展示所在原文件的行号
egrep–>grep -E 不使用反义符的情况下使用
egrep -n ‘o+o’ filename +与“”星号类似，但是+表示为1-n次出现，不包含0次
egrep -n ‘o?t’ filename “？”表示问号前面出现的字符为0或者1次，有则出现，没有这只匹配后面的字符出现
egrep -n ‘roo|nologin’ filename 同时筛选roo与nologin字符所在的行
cat aa.txt|egrep   '^[a-z]+$'  查看全是英文的行
cat aa.txt|egrep   '^[0-9]+$'  查看全是数字的行
```

###### 对 Linux 服务器网卡进行带宽限制的办法：


限制网卡带宽

限制 eth0 网卡的带宽为 500kbit：

```bash
tc qdisc add dev eth0 root tbf rate 500kbit latency 50ms burst 15kb
```

限制 eth0 网卡的带宽为 1mbit：

```bash
tc qdisc add dev eth0 root tbf rate 1mbit burst 10kb latency 70ms peakrate 2mbit minburst 1540
```

解除 eth0 网卡的带宽限制

```bash
tc qdisc del dev eth0 root
```

列出 eth0 网卡已有的策略

```bash
tc -s qdisc ls dev eth0
```

###### 实时监控网卡流量

```shell
#!/bin/bash
#echo -e "\033[31m  In --------- Out \033[0m"
while true;do
   echo -e "\033[34m ------------------------------------------- \033[0m"
   echo -e "\033[34m |\033[0m    \033[35m 网卡名称     进流量     出流量\033[0m      \033[34m| \033[0m"
   echo -e "\033[34m ------------------------------------------- \033[0m"

   for  NIC in $(ifconfig |awk '{print $1}'|grep -v "lo:" | grep ':' |awk -F ':' '{print $1}')
   ##for NIC in eth0   tun0  #此处为自定义单个或多个网卡名称
   do
    OLD_IN=$(awk '$0~"'$NIC'"{print $2}' /proc/net/dev)
    OLD_OUT=$(awk '$0~"'$NIC'"{print $10}' /proc/net/dev)
    sleep 1
    NEW_IN=$(awk '$0~"'$NIC'"{print $2}' /proc/net/dev)
    NEW_OUT=$(awk '$0~"'$NIC'"{print $10}' /proc/net/dev)
    IN=$(printf "%.1f%s" "$((($NEW_IN-$OLD_IN)/1024))" "KB/s")
    OUT=$(printf "%.1f%s" "$((($NEW_OUT-$OLD_OUT)/1024))" "KB/s")
    echo -e  "\033[32m 网卡名称:$NIC \033[0m    \033[31m $IN \033[0m     \033[33m $OUT  \033[0m"
    #sleep 1
  done
done

```
